Apa itu HTML ?

HTML (HyperText Markup Language) adalah bahasa markup standar untuk membuat halaman web. Ini menyusun konten halaman web menggunakan elemen seperti judul, paragraf, tautan, gambar, dan banyak lagi.

Struktur HTML Dasar

Dokumen HTML biasanya mencakup:

  • DOCTYPE: Menentukan versi
  • html: Elemen root yang berisi seluruh dokumen.
  • head: Berisi metadata dan tautan ke style css atau script
  • body: Berisi konten halaman web yang terlihat.

Apa itu HTML Form?

Form HTML digunakan untuk mengumpulkan inputan pengguna dan mengirimkannya ke server. Ini penting untuk membuat aplikasi web interaktif, mengaktifkan fungsi seperti registrasi pengguna, login, dan pengiriman data.

Elemen Bentuk Umum pada Form

Form biasanya mencakup berbagai elemen inputan, seperti:

  • Text Input: Digunakan untuk input teks pendek seperti nama dan nama pengguna.
  • Password Input: Menyembunyikan teks yang dimasukkan, ideal untuk kata sandi.
  • Radio Buttons: Memungkinkan pengguna untuk memilih satu opsi dari serangkaian.
  • Checkboxes: Memungkinkan pengguna memilih beberapa opsi.
  • Dropdown Menus: Memberikan daftar opsi dalam bentuk yang ringkas.
  • Submit Button: Mengirim data formulir ke server.
