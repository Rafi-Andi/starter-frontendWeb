HTML

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur dan konten halaman web. HTML merupakan dokumen standar yang digunakan untuk mendesain halaman web.

CSS

Bahasa yang digunakan untuk mendesain tampilan halaman web, termasuk warna, tata letak, dan animasi.

CSS

CSS adalah singkatan dari Cascading Style Sheets yang digunakan untuk mengatur tampilan dan desain halaman web. CSS berfungsi untuk mengatur elemen HTML, seperti warna, font, tata letak halaman, dan spasi antar paragraf.
